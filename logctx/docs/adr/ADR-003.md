# ADR-003: slog Integration

## Status: Accepted

**Date:** 2026-04-05  
**Author:** Phenotype Engineering Team  
**Project:** logctx

---

## Context

The logctx library must integrate with Go's standard structured logging package.

---

## Decision

We will use **Go 1.21+ slog as the only supported logger**:

```
┌────────────────────────────────────────────────────────────────┐
│              slog Integration                                   │
│                                                                 │
│  import "log/slog"                                             │
│                                                                 │
│  // Store in context                                           │
│  func WithLogger(ctx context.Context, logger *slog.Logger) context.Context {│
│      return context.WithValue(ctx, loggerKey, logger)          │
│  }                                                             │
│                                                                 │
│  // Retrieve from context                                      │
│  func From(ctx context.Context) *slog.Logger {                 │
│      // ...                                                    │
│  }                                                             │
│                                                                 │
│  Rationale:                                                    │
│  - Standard library, zero dependencies                        │
│  - Future-proof                                                │
│  - Structured logging by default                               │
│                                                                 │
└────────────────────────────────────────────────────────────────┘
```

---

## Consequences

### Positive

- **Standard**: Uses official library
- **No deps**: Zero external dependencies

### Negative

- **Go 1.21+ required**: Newer Go version
- **No zap/zerolog**: Must adapt if needed
