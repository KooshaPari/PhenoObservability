# ADR-003: Multi-Environment Configuration

## Status
Accepted

## Context
Dashboards require environment-specific configuration:
- Different data sources per environment
- Variable refresh rates (faster in dev, slower in prod)
- Environment-specific alert thresholds
- Feature availability differences

## Decision
Implement environment-based configuration injection that:
1. Uses a base dashboard definition
2. Applies environment-specific overrides
3. Validates configuration completeness
4. Supports local development defaults

## Implementation

```go
type Config struct {
    Environment   string
    DataSources   map[string]DataSourceConfig
    Alerting      AlertingConfig
    Templating    TemplateConfig
}

func (d *Dashboard) ApplyConfig(cfg Config) *Dashboard {
    // Clone base dashboard
    clone := d.Clone()
    
    // Apply environment-specific data sources
    for _, panel := range clone.Panels {
        for _, target := range panel.Targets {
            if ds, ok := cfg.DataSources[target.Datasource]; ok {
                target.Datasource = ds.UID
            }
        }
    }
    
    return clone
}
```
